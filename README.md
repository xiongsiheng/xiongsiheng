## 🔬 Projects

- [**MLR**](https://github.com/xiongsiheng/MLR) (ICLR 2026): Enhancing language model reasoning with structured multi-level modeling

- [**LogicLLaMA**](https://github.com/gblackout/LogicLLaMA) (ACL 2024): Harnessing the power of large language models for natural language to first-order logic translation
